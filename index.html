Final Project

This project showcases a collection of images selected to demonstrate color, contrast, and visual balance across different screen sizes. Each image includes clear alternative text and captions to support accessibility for all users. The goal of this gallery is to present simple, clean visuals while practicing responsive design techniques such as fluid layouts, flexible images, and proper use of media queries.

The page is built using a mobile-first approach, meaning the layout is optimized for small screens before being enhanced for larger displays. As the screen width increases, the gallery automatically adjusts from a single-column view to a multi-column grid, ensuring an enjoyable viewing experience on phones, tablets, laptops, and desktop monitors. This project also includes accessibility checks, image descriptions, and responsive styling to meet modern web design standards.
