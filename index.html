Final Project

This project presents a responsive image gallery featuring well-known characters from the One Piece universe. Each image includes descriptive alternative text and captions to support accessibility, while the layout is designed to adapt smoothly across different screen sizes. The purpose of this site is to demonstrate how visual content—such as character portraits—can be displayed effectively using modern web design principles.

Built with a mobile-first approach, the page begins as a simple single-column layout for small screens and gradually expands into multiple columns on tablets and desktops. As the viewport widens, the gallery automatically reorganizes to provide a cleaner and more efficient viewing experience. This project highlights the use of flexible images, fluid measurements, and media queries to ensure that fans can enjoy the character gallery on any device, while also meeting accessibility and responsiveness standards.
